Bash commands I use

I hope you didn't come here looking for a nicely formated easy to use guide to bash. This is just going to be a list of the bash commands I use and what they do. I may organsie it one day. Enjoy.

Key:
Orange things are code to copy
Teal things are hyperlinks
Thing in <pointy brackets> should be replaced with a filename, username ect. 

List how many files in each directory recursivley: du -a | cut -d/ -f2 | sort | uniq -c | sort -nr

List amount of things in a directory: du -hs

List the size of files: ls -lh
